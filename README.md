# online-complier
An online complier and editor for programming languages.

### Supported Languages

* Javascript
* C++
* Haskell
* JSON (editor only)
* CSS (editor only)
* HTML (editor only)
* Markdown (editor only)

<br>

### System Requirements
<hr>

- #### Docker (version 20.10.5) (TODO: Test other versions) ❌
- #### docker-compose (version 1.28.0 or higher)

<br>
